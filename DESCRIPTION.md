### Key Features
- 📊 Advanced Admin Dashboard
- 🔒 Secure Authentication
- 💳 Razorpay Integration
- 📱 Responsive Design
- 📈 Real-time Analytics

### Tech Stack
TypeScript, React, Node.js, Express, MongoDB, Redux Toolkit, SCSS
